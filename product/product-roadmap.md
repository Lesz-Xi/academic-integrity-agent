## Navigation Sections

This document defines the primary sections for the application shell navigation.

---

## Primary Navigation

### 1. Generator (Home)
**Path:** `/`
**Description:** Main AI content generation interface with mode selection.
**Icon:** `Wand2`
**Access:** All authenticated users

### 2. Drafting Canvas
**Path:** `/editor`
**Description:** Full-screen writing environment with real-time telemetry capture.
**Icon:** `FileText`
**Access:** All authenticated users
**Status:** Beta

### 3. Certificates
**Path:** Modal
**Description:** View and download attestation certificates for completed work.
**Icon:** `Award`
**Access:** All authenticated users

### 4. Defense Toolkit
**Path:** Modal
**Description:** Access forensic evidence and timeline for academic integrity disputes.
**Icon:** `ShieldAlert`
**Access:** Premium users only

---

## Secondary Navigation

### History
**Location:** Sidebar
**Description:** Recent generations grouped by mode
**Behavior:** Collapsible disclosure panels

### User Menu
**Location:** Sidebar footer
**Contents:**
- User avatar and email
- Premium status indicator
- Upgrade prompt (for free users)
- Theme toggle
- Sign out

---

## Section Priority (Mobile)

On mobile devices, navigation should prioritize:
1. New Generation button
2. History access
3. User menu

Drafting Canvas, Certificates, and Defense are secondary and accessible via menu.
